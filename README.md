# DI-Juego
Hundir la Flota Juego hecho con Java Swing ...

Desarrollo del famosos juego Hundir la Flota desarollado en el lenguaje de programacion Java, y la parte gráfica de este esta desarrollada
Java Swing. El juego trata de seleccionar el barco que desea poner y clicar donde lo quieres poner, segun el barco que selecciones se pone
un color u otro.

Barco1:Color Amarillo
Barco2:Color Verde
Barco3:Color Rosa 
Barco4:Color Naranja

Clicas en el turno maquina genera una tirada , y tu dices el resultado de esa tirada. 

Para ello tienes que pulsar el boton adecuado ya sea

-Agua 
-Tocado
-Hundido
